# Reduced Motion Hero Demo

A React + GSAP demo that shows how animations can adapt to the user's `prefers-reduced-motion` setting.

## Demo

Coming soon

## Features

- Detects `prefers-reduced-motion`
- Switches hero animation based on motion preference
- Provides a manual toggle for previewing reduced motion
- Uses GSAP with `useGSAP`
- Adjusts hover animation for reduced motion

## Tech Stack

- React
- Vite
- GSAP
- SCSS

## Notes

This demo focuses on reducing motion while keeping the visual experience clear and accessible.
